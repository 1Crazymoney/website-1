Install

Install IPFS on your machine

After you have installed IPFS, get started adding files to the distributed web!

ipfs init # create ipfs local config file

ipfs daemon # start the ipfs deamon process

ipfs cat # show ipfs object data

ipfs get # download ipfs objects

ipfs ls # list links from an object

ipfs refs # list hashes of links from an object

Is the command line not your jam? Download ipfs-desktop!
